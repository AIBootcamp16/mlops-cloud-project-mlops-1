# 파일 위치
```text
/infrastructure/docker/docker-compose.yml
```

# 해당 위치에서 실행
```text
docker compose up -d
```

# 전체 파이프라인을 간단히 구성해놓았습니다.
```text
covid-analysis 서버 접근 
ex) docker exec -it dev-covid-analysis bash

혹은 주피터랩으로 실행
localhost:8888

실행하면 폴더 내부에
notebooks 접근
04_full_ml_pipeline.ipynb 접근 후 해당 코드 실행하게 되면
데이터 수집 > 전처리 > 피처 > 간단한 모델링 진행하여 

localhost:5000  접근하여 mlflow ui 확인가능합니다.

참고하셔서 개발을 하셔도되고 본인이 구성하신 코드들을 실행해보셔도 될것같습니다.

저장공간이나 파라미터 저장하는 것은 미리 구성해놓은 모듈들을 확인해보시면 될것같습니다.

모듈들은 각각 src 내부에 
데이터 수집:data_collection
전처리:data_processing
피쳐:feature_engineering
모델링: modeling
전체 파이프라인: pipline
기본 유틸: utils
설정:config 
참고하시면 될것같습니다.
```
